# Homepage Redesign Proposal

## Design Goals
- Modern, professional B2B aesthetic
- Clear value proposition
- Trust-building elements
- Easy access to login
- Visual interest without being flashy

## Layout Structure

### Hero Section
- **Large headline**: Clear value proposition
- **Subheadline**: Supporting description
- **Primary CTA**: Prominent login button
- **Visual element**: Subtle background pattern or gradient
- **Trust indicators**: Optional badges or stats

### Features Section (Optional)
- **Three key features**:
  - Live inventory access
  - Tiered pricing
  - Easy ordering
- **Icons or illustrations** for each feature

### Footer
- **Simple footer** with:
  - Contact information
  - Privacy/terms links (if applicable)
  - Copyright

## Design Principles
1. **Minimal but impactful**: Clean design with strategic use of space
2. **Professional**: B2B-appropriate, not consumer-focused
3. **Fast loading**: Optimized images and minimal dependencies
4. **Accessible**: Clear contrast, readable fonts
5. **Brand consistency**: Matches portal's dark theme
